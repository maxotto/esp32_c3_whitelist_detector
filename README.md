# О проекте

Проект предназначен для визуального отображения состояния Интернета в зависимости от доступности некоторых сайтов.

Основная задача - отлавливать периоды, когда РКН включает так называемые "белые списки". Актуально, в основном, при подключении к сети через мобильного оператора. Но всяко может быть.

Устройство раз в минуту пингует 3 сайта:

1. google.com;
2. kp40.ru;
3. dzen.ru.

На основании результатов пинга делается один из выводов:

1. Все в норме;
2. Доступен только российский сегмент Интернета;
3. Включены "белые списки";
4. Интернета нет вообще.

В соответствии с этим все 4 светодиода ленты загораются одним из цветов:
1. зеленый;
2. желтый;
3. оранжевый;
4. красный.

# Hardware
Esp32 c3 super mini. Для лучшей связи с сетью по WiFi применена антеннка из проволочки длиной 62,5 мм.

# Ограничения и недостатки

1. Имя сети и пароль WiFi жестко зашиты в коде;
2. Список сайтов для проверки жестко зашит в коде;
3. Временные константы тоже жестко прописаны в коде.
4. Варианты визульного отображения с учетом 4 адресных светодиодов могли бы быть и поинтереснее ;)
